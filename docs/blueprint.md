# **App Name**: AI Recruiter Assistant

## Core Features:

- Job Category Management: Manage job categories (e.g., IT, Sales, Healthcare) through a CRUD interface in the admin panel.
- Job Management: Define and manage specific jobs within each category, including required skills and soft skills, via the admin panel.
- Interview Prompt Management: Create, update, and manage interview prompts tailored to each job in the admin panel. Write and adjust the text that will serve as a tool for the AI during interviews.
- AI Interview Simulation: Simulate job interviews using AI based on predefined job prompts. Allow the AI to adapt interview questions based on candidate responses.
- Data Storage for Categories, Jobs and Prompts: Backend storage for job categories, specific jobs, and interview prompts, to be managed by the admin interface and used in the AI interview simulation.
- REST API for Data Access: Provide a REST API for the application and admin interface to securely access and manage stored data.

## Style Guidelines:

- Primary color: HSL(210, 60%, 50%) - A professional blue to inspire confidence and trust.
- Background color: HSL(210, 20%, 95%) - Light, desaturated blue for a clean, unobtrusive background.
- Accent color: HSL(180, 60%, 40%) - A contrasting teal to highlight key interactive elements.
- Headline font: 'Space Grotesk', a sans-serif with a modern and techy feel.
- Body font: 'Inter', a grotesque-style sans-serif for a clean, neutral, readable UI.
- Use clean, professional icons related to job categories and functions. Icons should be simple and easily recognizable.
- Implement a clean, intuitive layout for both the admin interface and the application. Focus on usability and clear navigation.
